Пример обнуляемого типа

В этом примере демонстрируется использование обнуляемых типов. Данная функция позволяет типам значений быть неинициализированными или пустыми, схожее состояние с ссылочными типами со значением null.

Примечание по безопасности Данный код примера предоставлен для иллюстрирования концепции и не должен использоваться в приложениях или веб-узлах, так как он не демонстрирует опыт написания безопасного кода. Microsoft не несет ответственность за случайные или косвенные повреждения, вызванные применением кода примера не по назначению.

Построение и выполнение примеров обнуляемого типа в Visual Studio

  1. Откройте файл решения Nullable.sln, дважды щелкнув файл в проводнике Windows или щелкнув Открыть в меню Файл.

  2. В меню Отладка щелкните Запуск без отладки.

Построение и выполнение примеров обнуляемого типа из командной строки

  1. С помощью команды Изменить каталог (cd) перейдите в каталог обнуляемого типа.

  2. Введите следующие данные: 

    cd Basics
csc Basics.cs
Basics
  3. Введите следующие данные:
     
    cd ..\Boxing
csc Boxing.cs
Boxing
  4. Введите следующие данные:
     
    cd ..\Operator
csc Operator.cs
Operator
