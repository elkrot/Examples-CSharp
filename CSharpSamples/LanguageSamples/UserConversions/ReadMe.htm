Пример преобразований, определяемых пользователем

Данный пример иллюстрирует определение преобразований из классов или структур и наоборот, а также использование данных преобразований. Дополнительные сведения см. в разделе Операторы преобразований (Руководство по программированию на C#).

Данный код примера предоставлен для иллюстрирования концепции и не должен использоваться в приложениях или веб-узлах, так как он не демонстрирует опыт написания безопасного кода. Построение и выполнение примеров преобразований, определяемых пользователем, в Visual Studio

  1. В Обозревателе решений щелкните правой кнопкой проект Conversion1 и выберите Установить как автозагружаемый проект.

  2. В меню Отладка щелкните Запуск без отладки.

  3. Повторите предыдущие шаги для Conversion2.

Построение и выполнение примеров преобразований, определяемых пользователем, из командной строки

  1. С помощью команды Изменить каталог перейдите в каталог Conversion1.

  2. Введите следующие данные: 

    csc conversion.cs
conversion

  3. С помощью команды Изменить каталог перейдите в каталог Conversion2.

  4. Введите следующие данные: 

    csc structconversion.cs
structconversion
