Пример структур

Данный пример иллюстрирует синтаксис и использование структур. Пример также иллюстрирует важные различия между классами и структурами. Дополнительные сведения см. в разделе Объекты, классы и структуры (Руководство по программированию на C#).

Построение и выполнение примеров структур в Visual Studio

  1. В Обозревателе решений щелкните правой кнопкой проект Struct1 и выберите Установить как автозагружаемый проект.

  2. В меню Отладка щелкните Запуск без отладки.

  3. Повторите предыдущие шаги для Struct2.

Построение и выполнение примеров структур из командной строки

  1. С помощью команды Изменить каталог перейдите в каталог Struct1.

  2. Введите следующие данные: 

    csc struct1.cs
struct1

  3. С помощью команды Изменить каталог перейдите в каталог Struct2.

  4. Введите следующие данные: 

    csc struct2.cs
struct2
