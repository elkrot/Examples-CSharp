Образец безопасности

Данный образец демонстрирует процедуру изменения разрешений безопасности посредством классов и атрибутов безопасности. Дополнительные сведения см. в разделе Безопасность (Руководство по программированию на C#).

Примечание по безопасности

Данный образец кода предоставлен для иллюстрирования концепции и не должен использоваться в приложениях или веб-сайтах, так как он не демонстрирует опыт написания безопасного кода. Построение и запуск образцов безопасности в Visual Studio

1.      В окне Обозреватель решений щелкните правой кнопкой мыши проект Security и выберите Назначить запускаемым проектом.

2.      В меню Отладка выберите Запуск без отладки.

Построение и запуск образцов безопасности из командной строки

1.      Используйте команду Сменить каталог для перехода в каталог Security.

2.      Введите следующие данные: 

csc Security.cs
Security
