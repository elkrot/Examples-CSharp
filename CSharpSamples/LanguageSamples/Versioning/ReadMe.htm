Пример отслеживания версий

[Этот раздел из предварительной документации может быть изменен в будущих выпусках. Пустые разделы включены как прототипы.]

Данный пример демонстрирует отслеживание версий в C# посредством использования ключевых слов переопределение и новый. Отслеживание версий позволяет поддерживать совместимость базы и возникающих производных классов. Дополнительную информацию см. в разделе Отслеживание версий с ключевыми словами Переопределение и Новый (Руководство по программированию на C#).

Примечание по безопасности Данный код примера предоставлен для иллюстрирования концепции и не должен использоваться в приложениях или веб-узлах, так как он не демонстрирует опыт написания безопасного кода.

Построение и выполнение примеров отслеживания версий в Visual Studio

  1. Откройте решение (Versioning.sln).

  2. В меню Отладка щелкните Запуск без отладки.

Построение и выполнение примеров отслеживания версий из командной строки

  • Введите следующие данные в командную строку: 

    csc versioning.cs
