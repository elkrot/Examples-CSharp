Пример явной реализации интерфейса

Данный пример демонстрирует способ явной реализации членов интерфейса и способ доступа к данным членам из экземпляров интерфейса. Дополнительную информацию см. в разделе Интерфейсы (Руководство по программированию на C#).

Построение и выполнение примеров явной реализации интерфейса в Visual Studio

  1. В Обозревателе решений щелкните правой кнопкой проект ExplicitInterface1 и выберите Установить как автозагружаемый проект.

  2. В меню Отладка щелкните Запуск без отладки.

  3. Повторите предыдущие шаги для ExplicitInterface2.

Построение и выполнение примеров явной реализации интерфейса из командной строки

  1. С помощью команды Изменить каталог перейдите в каталог ExplicitInterface1.

  2. Введите следующие данные: 

    csc explicit1.cs
explicit1

  3. С помощью команды Изменить каталог перейдите в каталог ExplicitInterface2.

  4. Введите следующие данные: 

    csc explicit2.cs
explicit2
