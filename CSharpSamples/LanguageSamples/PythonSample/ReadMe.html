В этом образце демонстрируется порядок интеграции C# и Python с использованием нового динамического компонента C# 4.0. Для запуска этого образца необходимо установить приложение IronPython 2.0, загружаемое отдельно. Этот образец также может работать с более поздними версиями IronPython. После установки IronPython необходимо обновить этот образец, заменив сборки IronPython, IronPython.Modules, Microsoft.Scripting и Microsoft.Scripting.Core в разделе «Ссылки». В обозревателе решений щелкни те правой кнопкой мыши узел Ссылки. Удалите сборки, перечисленные выше. Выберите команду Добавить ссылку. Перейдите в каталог, в котором установлено приложение IronPython, и добавьте удаленные сборки. По умолчанию приложение IronPython устанавливается в каталог Program Files, в котором также находятся необходимые сборки. После обновления раздела «Ссылки» программа должна компилироваться и запускаться без ошибок.
