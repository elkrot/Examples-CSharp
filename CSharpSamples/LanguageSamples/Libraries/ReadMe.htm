Пример библиотек

В этом примере иллюстрируется создание и использование DLL в C#.

Построение и выполнение примеров библиотек в Visual Studio

  1. В Обозревателе решений щелкните правой кнопкой проект FunctionTest и выберите Установить как автозагружаемый проект.

  2. В Обозревателе решений щелкните правой кнопкой проект FunctionTest и выберите Свойства.

  3. Откройте папку Свойства конфигурации и щелкните Отладка.

  4. Для свойства Аргументы командной строки введите значение 3 5 10.

  5. Нажмите кнопку ОК.

  6. В меню Отладка щелкните Запуск без отладки. Это приведет к автоматическому построению библиотеки в Функциях и выполнению программы.

Построение и выполнение примеров библиотек из командной строки

  1. С помощью команды Изменить каталог перейдите в каталог Функции.

  2. Введите следующие данные: 

    csc /target:library /out:Functions.dll Factorial.cs DigitCounter.cs

  3. С помощью команды Изменить каталог перейдите в каталог FunctionTest.

  4. Введите следующие данные: 

    copy ..\Functions\Functions.dll .
csc /out:FunctionTest.exe /R:Functions.DLL FunctionClient.cs 
FunctionTest 3 5 10
