Пример поточной обработки

Данный пример демонстрирует следующие методики поточной обработки. Дополнительные сведения см. в разделе Поточная обработка (Руководство по программированию на C#).

  • Создание, запуск и завершение потока

  • Использование пула потоков

  • Синхронизация и взаимодействие потока

Примечание по безопасности

Данный код примера предоставлен для иллюстрирования концепции и не должен использоваться в приложениях или веб-узлах, так как он не демонстрирует опыт написания безопасного кода. Построение и выполнение примеров поточной обработки в Visual Studio

  1. В Обозревателе решений щелкните правой кнопкой проект ThreadStartStop и выберите Установить как автозагружаемый проект.

  2. В меню Отладка щелкните Запуск без отладки.

  3. Повторите предыдущие шаги для ThreadPool и ThreadSync.

Построение и выполнение примеров поточной обработки из командной строки

  1. С помощью команды Изменить каталог перейдите в каталог Threads.

  2. Введите следующие данные: 

    cd ThreadStartStop
csc ThreadStartStop.cs
ThreadStartStop

  3. Введите следующие данные: 

    cd ..\ThreadPool
csc ThreadPool.cs
ThreadPool

  4. Введите следующие данные: 

    cd ..\ThreadSync
csc ThreadSync.cs
ThreadSync
