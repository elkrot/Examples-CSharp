Пример небезопасного кода

Данный пример демонстрирует использование неуправляемого кода (код с указателями) в C#.

Примечание по безопасности

Данный код примера предоставлен для иллюстрирования концепции и не должен использоваться в приложениях или веб-узлах, так как он не демонстрирует опыт написания безопасного кода. Microsoft не несет ответственность за случайные или косвенные повреждения, вызванные применением кода примера не по назначению.

Построение и выполнение примеров небезопасного кода в Visual Studio

  1. В Обозревателе решений щелкните правой кнопкой проект FastCopy и выберите Установить как автозагружаемый проект.

  2. В меню Отладка щелкните Запуск без отладки.

  3. В Обозревателе решений щелкните правой кнопкой проект ReadFile и выберите Установить как автозагружаемый проект.

  4. В Обозревателе решений щелкните правой кнопкой проект ReadFile и выберите Свойства.

  5. Откройте папку Свойства конфигурации и щелкните Отладка.

  6. Для свойства Аргументы командной строки введите значение ..\..\ReadFile.cs.

  7. Нажмите кнопку ОК.

  8. В меню Отладка щелкните Запуск без отладки.

  9. В Обозревателе решений щелкните правой кнопкой проект PrintVersion и выберите Установить как автозагружаемый проект.

  10. В меню Отладка щелкните Запуск без отладки.

Построение и выполнение примеров небезопасного кода из командной строки

  1. С помощью команды Изменить каталог перейдите в каталог «Unsafe».

  2. Введите следующие данные: 

    cd FastCopy
csc FastCopy.cs /unsafe
FastCopy

  3. Введите следующие данные: 

    cd ..\ReadFile
csc ReadFile.cs /unsafe
ReadFile ReadFile.cs

  4. Введите следующие данные: 

    cd ..\PrintVersion
csc PrintVersion.cs /unsafe
PrintVersion
