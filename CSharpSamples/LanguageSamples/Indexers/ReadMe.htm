Пример индексаторов

Данный пример содержит сведения по объявлению индексаторов классами C# с целью предоставления доступа по типу массивов к классам.

Построение и выполнение примеров индексаторов в Visual Studio

  1. В Обозревателе решений щелкните правой кнопкой проект индексаторов и выберите Свойства

  2. Откройте папку Свойства конфигурации и щелкните Отладка.

  3. Для свойства Аргументы командной строки введите значение ..\..\Test.txt.

  4. Нажмите кнопку ОК.

  5. В меню Отладка щелкните Запуск без отладки.

Построение и выполнение примеров индексаторов из командной строки

  1. Для компилирования программы примера введите следующий текст в командную строку: 

    csc indexer.cs

  2. Программа примера осуществляет реверс байтов в файле, предоставленном в качестве аргумента командной строки. Например, для выполнения реверса байтов в Test.txt и просмотра результатов требуется выполнить перечисленные далее команды: 

    indexers Test.txt
type Test.txt

  3. Для возврата файла, для которого был выполнен реверс, в нормальное состояние требуется повторно выполнить программу с тем же файлом.
