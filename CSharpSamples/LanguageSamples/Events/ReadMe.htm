Пример событий

Данный пример содержит сведения по объявлению, вызову и настройке событий в C#. Дополнительные сведения см. в разделе События (Руководство по программированию на C#).

Построение и выполнение примеров событий в Visual Studio

  1. В Обозревателе решений щелкните правой кнопкой проект Events1 и выберите Установить как автозагружаемый проект.

  2. В меню Отладка щелкните Запуск без отладки.

  3. Повторите предыдущие шаги для Events2.

Построение и выполнение примеров событий из командной строки

  1. С помощью команды Изменить каталог перейдите в каталог Events1.

  2. Введите следующие данные: 

    csc events1.cs
events1

  3. С помощью команды Изменить каталог перейдите в каталог Events2.

  4. Введите следующие данные: 

    csc events2.cs
events2
