Пример свойств

Данный пример иллюстрирует свойства в качестве неотъемлемой части языка программирования C#. Демонстрируются объявление и использование свойств. Дополнительные сведения см. в разделе Свойства (Руководство по программированию на C#).

Построение и выполнение примеров свойств в Visual Studio

  1. В Обозревателе решений щелкните правой кнопкой персональный проект и выберите Установить как автозагружаемый проект.

  2. В меню Отладка щелкните Запуск без отладки.

  3. Повторите предыдущие шаги для shapetest.

Построение и выполнение примеров свойств из командной строки

  1. С помощью команды Изменить каталог перейдите в персональный каталог.

  2. Введите следующие данные: 

    csc person.cs
person

  3. С помощью команды Изменить каталог перейдите в каталог «shapetest».

  4. Введите следующие данные: 

    csc abstractshape.cs shapes.cs shapetest.cs
shapetest
