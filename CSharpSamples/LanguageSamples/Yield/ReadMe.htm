Пример возврата

Данный пример демонстрирует создание класса списка, реализующего IEnumerable<int> и ключевое слово возврат с целью включения итерации foreach над содержимым списка. Заданы два свойства, одно свойство возвращает нечетные номера, второе свойство возвращает четные номера.
