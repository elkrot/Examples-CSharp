Пример перегрузки оператора

Данный пример демонстрирует перегрузку операторов классами, заданными пользователем. Дополнительные сведения см. в разделе Операторы C#.

Построение и выполнение примеров перегрузки оператора в Visual Studio

  1. В Обозревателе решений щелкните правой кнопкой проект Complex и выберите Установить как автозагружаемый проект.

  2. В меню Отладка щелкните Запуск без отладки.

  3. Повторите предыдущие шаги для Dbbool.

Построение и выполнение примеров перегрузки оператора из командной строки.

  1. С помощью команды Изменить каталог перейдите в каталог Complex.

  2. Введите следующие данные: 

    csc complex.cs
complex

  3. С помощью команды Изменить каталог перейдите в каталог Dbbool.

  4. Введите следующие данные: 

    csc dbbool.cs
dbbool
