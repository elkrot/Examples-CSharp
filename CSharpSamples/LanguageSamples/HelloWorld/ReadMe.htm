Пример Hello World

Построение и выполнение примеров Hello World в Visual Studio.

  1. Откройте решение (HelloWorld.sln).

  2. В Обозревателе решений щелкните правой кнопкой проект HelloWorld1 и выберите Установить как автозагружаемый проект.

  3. В меню Отладка щелкните Запуск без отладки.

  4. Нажмите любую клавишу для закрытия HelloWorld1.

  5. В Обозревателе решений щелкните правой кнопкой проект HelloWorld2 и выберите Установить как автозагружаемый проект.

  6. В меню Отладка щелкните Запуск без отладки.

  7. Нажмите любую клавишу для закрытия HelloWorld2.

  8. В Обозревателе решений щелкните правой кнопкой проект HelloWorld3 и выберите Установить как автозагружаемый проект.

  9. В Обозревателе решений щелкните правой кнопкой проект HelloWorld3 и выберите Свойства.

  10. Откройте папку Свойства конфигурации и щелкните Отладка.

  11. В свойстве Аргументы командной строки введите значение «A B C D» и нажмите OK.

  12. В меню Отладка щелкните Запуск без отладки.

  13. Нажмите любую клавишу для закрытия HelloWorld3.

  14. В Обозревателе решений щелкните правой кнопкой проект HelloWorld4 и выберите Установить как автозагружаемый проект.

  15. В меню Отладка щелкните Запуск без отладки.

  16. Нажмите любую клавишу для закрытия HelloWorld4.

Построение и выполнение примеров Hello World из командной строки.

  1. С помощью команды Изменить каталог перейдите в каталог HelloWorld.

  2. Введите следующие данные: 

    cd HelloWorld1
csc Hello1.cs
Hello1

  3. Введите следующие данные: 

    cd ..\HelloWorld2
csc Hello2.cs
Hello2

  4. Введите следующие данные: 

    cd ..\HelloWorld3
csc Hello3.cs
Hello3 A B C D

  5. Введите следующие данные: 

    cd ..\HelloWorld4
csc Hello4.cs
Hello4
