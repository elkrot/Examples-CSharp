Пример универсальных шаблонов (C#)

Данный пример иллюстрирует создание пользовательского класса списка универсальных шаблонов с параметром одного типа и иллюстрирует реализацию IEnumerable<T> с целью предоставления итерации foreach по содержимому списка. Пример также содержит сведения о созданию кодом клиента экземпляра класса путем указания аргумента типа и сведения по обеспечению ограничениями параметра типа дополнительных операций для выполнения над аргументами типа.
