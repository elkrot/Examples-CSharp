Описываемые далее примеры были специально составлены для демонстрации возможностей разработки решений Office с помощью Visual Studio 2010.  Эти возможности могут быть использованы для работы с Office 2007, Office 2010 и последующими версиями. 

Для большей эффективности загрузите версии примеров, соответствующие используемой версии NET Framework.  Примеры размещаются в следующих подкаталогах:

\Office
    \FX40 - примеры, оптимизированные для выполнения с клиентским профилем .NET Framework 4.0  [По умолчанию для Visual Studio 2010]
    \FX35 - примеры, оптимизированные для выполнения с клиентским профилем .NET Framework 3.5; могут быть модифицированы для работы с 4.0 путем минимальных изменений, описанных ниже

Дополнительные сведения о различиях в разработке для Office в 3.5 и 4.0 см. в разделе:
Перенос решений Office на платформу .NET Framework 4
