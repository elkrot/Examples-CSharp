Примечание.

Данный пример работает только в Microsoft Office Outlook 2007 и последующих версиях.

В этом примере описывается порядок использования областей формы Microsoft Office Outlook для отображения всего содержимого статей из RSS-канала. Обычно в приложении Outlook отображаются только заголовки статей.

Во время просмотра элемента RSS в области чтения к его нижней части добавляется примыкающая область формы.

Во время просмотра элемента в инспекторе Outlook дополнительная область формы добавляется на отдельной странице элемента RSS.

Запуск примера

  1. Нажмите клавишу F5.

    Открывается приложение Outlook.

  2. Откройте элемент RSS. Дополнительные сведения см. в разделе «Чтение RSS-каналов».

    Примечание.

    Для выполнения примера требуется подписка на RSS-канал. Дополнительные сведения см. в разделе «Добавление RSS-канала».

  3. Выберите сообщение в списке.

    Сообщение отображается в области чтения. Полный текст статьи отображается в примыкающей области формы под сообщением.

  4. Дважды щелкните сообщение в списке.

    Сообщение открывается в окне инспектора.

  5. В группе Отобразить на ленте выберите команду Просмотреть статью.

    Полный текст статьи отображается на новой странице формы.

  6. В верхней части страницы выберите команду Поиск похожих тем.

    Открывается веб-сайт поиска Live Search, на котором отображаются результаты поиска, связанные с заголовком статьи.

  7. В верхней части страницы щелкните ссылку Скрыть результаты поиска.

    Результаты поиска скрываются.

Требования

Для выполнения этого примера требуются следующие компоненты:

  • Набор средств Visual Studio Tools для Office.

  • Microsoft Office Outlook 2007.

  • Подписка на RSS-канал. Сведения о получении подписки на RSS-канал см. в разделе «Добавление RSS-канала».

  • Включенная область чтения в приложении Outlook. Дополнительные сведения см. в разделе «Включение и отключение области чтения».

Демонстрации

В этом примере показано выполнение следующих задач:

  • Создание отдельной и примыкающей областей формы.

  • Использование управляемых элементов управления в области формы.

  • Доступ к экземпляру класса Microsoft.Office.Interop.Outlook.PostItem. Этот класс предоставляет элемент RSS, с помощью которого отображается область формы.

  • Использование экземпляра Microsoft.Office.Interop.Outlook.PostItem в обработчике событий Microsoft.Office.Tools.Outlook.FormRegionControl.FormRegionShowing для обработки области формы перед ее отображением в Outlook.
