Примечание.

Данный пример работает только в Microsoft Office Word 2007 и последующих версиях.

В данном примере демонстрируется привязка элементов управления содержимым в документе Word к полям в базе данных Microsoft Office Access. В учебной базе данных «Борей» каждый элемент управления в документе привязан к отдельному полю. Документ также отображает область действий, которая используется для перемещения по записям сотрудников, поиска определенных записей и сохранения изменений базы данных, сделанных в одном из элементов управления содержимым.

Запуск примера

  1. Нажмите клавишу F5.

    Убедитесь, что элементы управления содержимым в таблице отображают информацию о сотруднике по имени Nancy Davolio.

  2. В области действий нажмите кнопки Вперед и Назад для просмотра различных записей сотрудников.

    Убедитесь, что элементы управления содержимым отображают информацию выбранной в данный момент записи о сотруднике.

  3. В области действий введите число от 1 до 9 в текстовом поле Input ID (Ввод идентификатора) и нажмите кнопку Search by ID (Поиск по идентификатору).

    Убедитесь, что элементы управления содержимым отображают информацию записи выбранного идентификатора.

  4. Щелкните ячейку справа от ячейки Title (Должность) и введите новую должность для выбранного сотрудника.

  5. В области действий нажмите кнопку Submit change to Title (Сохранить изменение должности) для сохранения новой должности в базе данных.

  6. Откройте базу данных Борей.mdb в Access и убедитесь, что поле сотрудника Title (Должность) содержит новую должность.

Требования

Для выполнения этого примера требуются следующие приложения:

  • Набор средств Visual Studio Tools для Office.

  • Microsoft Office Word 2007.

  • Microsoft Office Access.

  • Учебная база данных «Борей» для Microsoft Office Access (содержится в примере).

Демонстрации

В этом примере показано выполнение следующих задач:

  • Привязка следующих элементов управления содержимым к полям базы данных:

    • PlainTextContentControl

    • PictureContentControl

    • DatePickerContentControl

  • Сохранение содержимого элемента управления содержимым с привязкой к данным в базу данных (этот процесс также называется двусторонней привязкой данных).

  • Использование панели действий.
