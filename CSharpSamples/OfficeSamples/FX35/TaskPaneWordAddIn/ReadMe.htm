Примечание.

Данный пример работает только в Microsoft Office Word 2007 и последующих версиях.

В этом примере описывается порядок создания пользовательской области задач для приложения Microsoft Office Word. Создаваемая область задач содержит текстовое поле и кнопку. После того как пользователь вводит текст в поле и нажимает кнопку, введенная строка добавляется в начало активного документа.

В этом примере область задач создается автоматически при загрузке надстройки. Функции отображения или скрытия области задач пользователем не реализованы. Обычно в надстройке реализуется элемент пользовательского интерфейса (например, кнопка на ленте), который используется для отображения и скрытия области задач. О том, как это сделать, см. в разделе «Пошаговое руководство. Синхронизация настраиваемой области задач с кнопкой на ленте».

Для создания области задач в этом примере используются классы CustomTaskPane и CustomTaskPaneCollection, доступные в наборе средств Visual Studio Tools для Office. Дополнительные сведения об использовании этих классов при создании областей задач см. в разделе «Общие сведения о настраиваемых областях задач».

Запуск примера

  1. Нажмите клавишу F5.

  2. Введите любой текст в текстовое поле области задач Contoso.

  3. Нажмите кнопку Вставить.

    Убедитесь, что введенный текст вставляется в начало активного документа.

Требования

Для выполнения этого примера требуются следующие приложения:

  • Набор средств Visual Studio Tools для Office.

  • Microsoft Office Word 2007.

Демонстрации

В этом примере показано выполнение следующих задач:

  • Создание и отображение настраиваемой области задач в надстройке Word.

  • Вставка текста, введенного в пользовательской области задач, в активный документ Word.
