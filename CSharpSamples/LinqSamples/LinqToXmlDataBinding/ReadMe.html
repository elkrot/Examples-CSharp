Пример привязки данных LINQ to Xml иллюстрирует привязку данных в WPF с помощью Linq To Xml.  Пример позволяет формировать поток видеоданных из Soapbox в MSN Videos и сохранять список избранного (Избранное).  Интерфейс имеет две вкладки, " Soapbox в MSN Video" и " Правка избранного".  Soapbox в MSN Video позволяет просматривать предварительно определенные видеосписки, также результаты поиска.  При необходимости видео можно добавить в список Избранное с помощью кнопки Добавить.  На вкладке Правка избранного можно обновить описание видео, а также удалить его из списка Избранное.  Для просмотра сведений по привязке данных элементов управления к потоковым XML-данным см. файл Window1.xaml.

Многие примеры LINQ, включая LINQ в «Борей», требуют наличия примера базы данных «Борей», расположенного в каталоге данных. По умолчанию каталог данных расположен в той же родительской папке, что и другие примеры LINQ. Это позволит автоматически определить путь к файлам базы данных. Для многих файлов примеров также требуется наличие проекта ObjectDumper.
