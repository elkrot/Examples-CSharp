Данный пример является кратким введением в блок System.Xml.Linq для знакомства пользователя с XML DOM API. Целью примера является иллюстрация конструирования предварительно определенного XML-документа с помощью LINQ to XML. Продемонстрированы два различных подхода. Первый подход: для сборки документа используется API, похожий на DOM XML (см. CreateDocumentVerbose()). Обратите внимание на различные опции для добавления узлов в документ. Второй подход, известный как функциональное конструирование, является более сжатым стилем, который позволяет достичь тот же результат (см. CreateDocumentConcise()). Пример также демонстрирует метод перемещения документа (см. DumpNode()). Обратите внимание на похожие свойства, предоставленные на узлах различных типов.

Для многих примеров LINQ требуется наличие XML-файлов и/или версии примера базы данных «Борей», расположенной в примере данных. При установке примера данных в ту же родительскую паку, что и примеры LINQ, путь к XML-файлам и файлу базы данных определяется автоматически. Для многих файлов примеров также требуется наличие проекта ObjectDumper.
