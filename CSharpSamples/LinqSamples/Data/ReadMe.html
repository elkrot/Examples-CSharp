Этот каталог данных используется несколькими другими примерами LINQ. Обратите внимание на копию базы данных с названием Northwind.mdf, которая находится в данном каталоге. Это пользовательская версия базы данных Борей, используемая многими продуктами корпорации Microsoft. Изменения в этой базе данных включают добавление новых сохраненных процедур и полей, разработанных для представления отдельных функций LINQ. Проект SampleQueries содержит вторую копию этой базы данных. Две копии базы данных «Борей» предоставлены, чтобы SampleQueries могли быть легко скопированы в качестве отдельных единиц из одного местоположения в другое. При перемещении других примеров, например, LinqToNorthwind, каталог Data необходимо также переместить вместе с примерами.

Среди других файлов в данном каталоге небольшие XML-файлы, используемые некоторыми другими примерами LINQ. Обратите внимание, что данные XML-файлы также дублируются в каталог SampleQueries по указанным выше причинам.

Необходимо, чтобы файлы в данном каталоге не имели атрибута только для чтения. Многие примеры не будут выполняться корректно, если данные файлы имеют атрибут только для чтения. В частности, могут возникнуть проблемы при подключении к базе данных «Борей», если отсутствуют права на запись и чтение Northwind.mdf.

Многие примеры LINQ требуют наличия XML-файлов и/или версии примера базы данных «Борей», расположенной в примере данных. При установке примера данных в ту же родительскую паку, что и примеры LINQ, путь к XML-файлам и файлу базы данных будет установлен автоматически. Для многим файлов примеров также требуется наличие проекта ObjectDumper.
