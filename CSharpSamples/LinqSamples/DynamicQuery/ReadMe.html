Данный пример демонстрирует методику динамического формирования операторов LINQ «на лету» во время выполнения. См. файл Dynamic Expressions.html в текущем каталоге.

При возникновении проблем с выполнением примеров баз данных см. примечание Соединение в корневом каталоге проекта примеров.

Для многих примеров LINQ требуется наличие XML-файлов и/или версии примера базы данных «Борей», расположенной в каталоге Data. По умолчанию каталог Data находится в той же родительской паке, что и другие примеры LINQ, поэтому путь к XML-файлам и файлу базы данных определяется автоматически. Для многих файлов примеров также требуется наличие проекта ObjectDumper.
