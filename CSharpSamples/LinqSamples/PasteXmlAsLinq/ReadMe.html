Данный проект является надстройкой Visual Studio, которая позволяют улучшить набор функций в интегрированной среде разработки, путем добавления пользователям возможности автоматического преобразования верных XML-данных в соответствующие операторы LINQ to XML. Версии экспресс-выпуска среды Visual Studio не поддерживают надстройки.

После сборки скопируйте файлы PasteXmlAsLinq.dll и PasteXmlAsLinq.Addins в каталог, который, возможно, потребуется создать, с именем:

..Documents\Visual Studio 2008\Addins

Потребуется перезапуск среды после копирования файлов.

Для просмотра функционирования надстройки требуется выбрать определенные XML-данные и вставить их в файл источника C#. В частности, перейдите в меню Правка и выберите элемент меню Вставить XML как XElement. Обратите внимание, что элемент не отобразится в меню при отсутствии данных в буфере обмена.

Далее представлены некоторые XML-данные для использования в качестве примера.

<mynode></mynode>

Результат вставки должен выглядеть следующим образом:

XElement xml = new XElement("mynode", "");

Для многих примеров LINQ требуется наличие XML-файлов и/или версии примера базы данных «Борей», расположенной в примере данных. При установке примера данных в ту же родительскую паку, что и примеры LINQ, путь к XML-файлам и файлу базы данных определяется автоматически. Для многих файлов примеров также требуется наличие проекта ObjectDumper.
