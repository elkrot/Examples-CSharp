Это библиотека, используемая некоторыми другими примерами LINQ.

Файл NorthwindMapping.DBML в данном проекте, генерирует пользовательские классы среды CLR, используемые для доступа к нижестоящей базе данных SQL Server.  Путем изменения свойств в конструкторе для NorthwindMapping.DBML можно изменить сгенерированные классы и обеспечить наличие точных требуемых отношений типов данных и объектов.  Результирующие классы можно просмотреть путем просмотра сгенерированного файла кода программной части NorthwindMapping.designer.cs.
