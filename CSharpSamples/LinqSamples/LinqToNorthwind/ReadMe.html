LINQ в «Борей» –это пример программы базы данных LINQ to SQL. В файле Samples.cs содержится пятнадцать примеров запросов LINQ to SQL, сохраненных в статических методах, с последовательными именами Sample1(), Sample2() и т. д. По умолчанию основной файл проекта Program.cs вызывает Sample1(). Отредактируйте Program.cs для вызова других примеров. Например, для вызова второго примера измените вызов Samples.Sample1() для чтения Samples.Sample2().

При возникновении проблем с выполнением примеров баз данных см. примечание Соединение в корневом каталоге проекта примеров.

Многие примеры LINQ, включая LINQ в «Борей», требуют наличия примера базы данных «Борей», расположенного в каталоге данных. По умолчанию каталог данных расположен в той же родительской папке, что и другие примеры LINQ. Это позволит автоматически определить путь к файлам базы данных. Для многих файлов примеров также требуется наличие проекта ObjectDumper.
