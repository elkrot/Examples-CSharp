Простой пример использования лямбда-выражений. Многие программисты захотят работать с выражениями запроса. Однако лямбда-выражения могут быть более полезны в некоторых обстоятельствах, а данный пример быстро ознакомит вас с возможностями.

Данный проект содержит 11 примеров. По умолчанию выполняется первый пример. Изменяя метод Main() в Program.cs, можно изменить код для выполнения одного из 10 примеров. Например, по умолчанию основными вызовами являются следующие вызовы.

        Samples.Sample1();

Измените данный код для вызова Sample2().

        Samples.Sample2();

Примеры имеют имена Sample1(), Sample2() и так далее до Sample10(). Сопровождая Sample9(), существует также метод Sample9a().
