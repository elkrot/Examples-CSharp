101 пример запроса

Данные проект содержит около 101 примера из перечисленных далее областей.

LINQ To Objects
LINQ To SQL
LINQ to DataSet
LINQ to Xml

Запустите проект и выберите тип примера LINQ, обзор которого требуется выполнить. Просмотрите опции и выполните каждый отдельный интересующий пример, как показано на Рисунке 1. Вывод примера и код, включенный в пример, будут отображаться в интерфейсе приложения.

При возникновении проблем с выполнением примеров баз данных см. примечание Соединение в корневом каталоге проекта примеров.


Рисунок 01. Пользователь перешел к примерам «Варианты использования XQuery» и выбрал тип примера «Поиск строки» для выполнения. Код в данном примере отображается рядом с верхней частью формы с правой стороны. Вывод отображается в нижней части с правой стороны.


Рисунок 02. Выполнение примера базы данных. Пользователь выбрал пример «LINQ to SQL». Код в данном примере отображается рядом с верхней частью формы с правой стороны, а вывод в нижней части с правой стороны. Нажмите кнопку Выполнить пример для выполнения каждого отдельного примера, включенного в проект.
