Это образец расширения Visual Studio 2010, созданного для использования вместе 
с новыми шаблонами проектов SharePoint.  В нем используются API расширяемости, 
входящие в комплект новых средств разработки SharePoint в составе Visual Studio 
2010. Это расширение предназначено для поддержки настраиваемого действия 
SharePoint.  Чтобы использовать данный образец по назначению, необходимо 
установить пакет SDK для Visual Studio 2010.  Его можно загрузить со страницы 
http://go.microsoft.com/fwlink/?LinkId=164562.

Решение содержит два проекта:
- Проект библиотеки классов, реализующий расширение.
- Проект VSIX для упаковки расширения в файл VSIX. Этот файл используется для 
  установки расширения.

Чтобы запустить образец, откройте в Visual Studio 2010 версию решения 
CustomActionProjectItem для Visual Basic или C# и нажмите клавишу F5. Будут 
выполнены следующие действия:
- Построение расширения на основе проекта ProjectItemDefinition.
- Создание пакета VSIX на основе проекта CustomActionProjectItem.
- Запуск экспериментального экземпляра Visual Studio, в котором устанавливается 
  пакет VSIX.

В экспериментальном экземпляре Visual Studio можно тестировать настраиваемое 
действие с помощью следующей процедуры:
- Создайте новый пустой проект SharePoint. Этот шаблон проекта становится 
  доступным, если в диалоговом окне «Создание проекта» в узле «SharePoint» 
  выбрать узел «2010». Используйте тот же язык программирования (Visual Basic 
  или C#), что и в открытом проекте CustomActionProjectItem.
- В окне «Мастер настройки SharePoint» введите URL-адрес сайта SharePoint на 
  локальном компьютере и нажмите кнопку «Готово».
- После создания проекта щелкните его узел в обозревателе решений правой 
  кнопкой мыши и выберите «Добавить | Новый элемент».
- В диалоговом окне «Добавление нового элемента» щелкните узел «SharePoint», 
  выберите элемент «Настраиваемое действие» и нажмите кнопку «Добавить».

Настраиваемое действие будет добавлено в проект. Можно выполнить следующие 
задачи:
- В обозревателе решений щелкните правой кнопкой мыши узел CustomAction и 
  выберите в контекстном меню команду «Конструктор настраиваемых действий». Это 
  команда добавляется в меню после установки расширения.
- В обозревателе решений выберите узел CustomAction и откройте окно «Свойства». 
  Убедитесь, что в окне свойств показано «Свойство настраиваемого действия». 
  Это свойство добавляется при установке расширения. 
- Измените содержимое файла Elements.xml в элементе проекта CustomAction и 
  нажмите клавишу F5, чтобы развернуть его на локальном сайте, который был 
  указан при создании проекта.
